How it works

01

Observe navigation

Monitors top-level navigations and user-initiated link clicks.

02

Score the domain

WHOIS age, redirects, reputation signals and heuristics.

03

Enforce policy

Warn, restrict or block based on risk thresholds.

What this extension does not do

This tool does not guarantee safety. Compromised legitimate websites, zero-day phishing campaigns and short-lived domains can bypass any heuristic system.

If you expect perfect detection, this is not the product for you.

Features

Automatic evaluation

No buttons. Decisions happen on navigation.

Per-domain overrides

Local allow/block rules always win.

Minimal data exposure

Only the hostname is transmitted.

Self-hostable backend

API keys stay on your server.

Install

  1. This is a developer-mode extension.
  2. Clone or download the repository.
  3. Enable developer mode in your browser.
  4. Load the extension folder.
cd backend
npm install
npm run dev
